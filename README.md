# TipCalculatorCompose
Tip App in Jetpack Compose, casual app


Features:

  Tip Calculation:

    - Add the price of check
    - Choose the percentage
    - Select how many customers

Tech:
- Jetpack Compose
- Material Design 3 (Beta)
- Custom Font (MPlusRounded1C)
